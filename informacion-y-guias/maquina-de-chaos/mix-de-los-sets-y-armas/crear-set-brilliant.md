# ▫️ Crear Set Brilliant

Para crear tu set Brilliant (Season 17) necesitaras los siguientes ítems :

* 1 parte del set Manticore  +15 +28 (Obligatorio)
* 1 Brilliant Soul Moru (Obligatorio)
* Bundle Jewel Bless +2 (Pack x30) (Obligatorio)
* Bundle Jewel Soul +2 (Pack x30) (Obligatorio)
* 1 Jewel Creation (Obligatorio)
* 1 Jewel Chaos (Obligatorio)
* Talismán of Luck máximo 10 **(Para aumentar el porcentaje de éxito)** **(Opcional)**

Debes hacer Click en el NPC **"Chaos Goblin"** e ir a la primer opción llamada **"Regular Combination"**

**¡MUCHA SUERTE!**

_**Nota: La imagen solo muestra el mix de 1 sola parte del set, tendrán que subir las otra piezas de igual forma.**_
